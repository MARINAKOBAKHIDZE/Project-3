PROJECT TITLE:
"Home Finder"
An Application that helps you find a home for rent or for purchase.
It gives you different price options to chose from.

INSALLATION 
Run npm init
    npm install, 
    npm install react react-dom --save,
    npm run dev.

TECHNOLOGIES USED
    HTML,
    CSS,
    JavaScript,
    Node,
    SQL.
